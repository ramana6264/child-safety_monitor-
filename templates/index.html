About This System

This emotion detection system uses a deep learning model trained on facial expressions to recognize human emotions in real-time. The system can detect the following emotions:

Emotion Classes

  • Angry shows signs of anger or frustration.
  • Fear shows signs of fear or anxiety.
  • Happy shows signs of happiness.
  • Sad shows signs of sadness.
  • Surprise shows signs of surprise.

Features

  • Real-time emotion detection from webcam.
  • Video upload analysis with live overlays.
  • Confidence scoring for each predicted emotion.
  • Multiple face detection and analysis.
  • Danger object alerts with escalation styling.

How to Use

  1. Open the upload tab to analyze a recorded clip.
  2. Open the webcam tab for continuous live detection.
  3. Watch the overlays, face counts, and emotion results update.
  4. Use the ESP status page to monitor attached-device location data.
